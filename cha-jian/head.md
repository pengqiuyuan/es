`elasticsearch-head`

安装`elasticsearch-head`，`/home/idatage/plugins`目录下

```
git clone git://github.com/mobz/elasticsearch-head.git
cd elasticsearch-head
sudo npm install
sudo nohup npm run start &
```

更新 `nodejs`

```
sudo apt-get install -y nodejs
nodejs -v
npm -v
sudo npm install -g n
sudo n latest
```
